# BeastAndBow - The Hunter's Essential Toolkit
screenshot.jpg

A lightweight and responsive Quality-of-Life addon for World of Warcraft: Classic Hunters.

BeastAndBow solves two core problems: constantly checking your inventory for ammunition/food, and quickly finding the nearest vendor in any zone.

## Installation & Usage

### Installation
1. Download the latest version from CurseForge (Recommended).
2. Extract the file into your `WoW\_classic_\Interface\AddOns` folder.

### Commands
| Command | Function |
| :--- | :--- |
| `/bab show` | Shows the main frame. |
| `/bab hide` | Hides the main frame. |
| `/bab mapdebug` | Displays verbose debug information in chat. |
